рҹ’Ўз®Җд»Ӣ

еңЁKubernetesдёӯпјҢDeploymentгҖҒStatefulSetе’ҢDaemonSetжҳҜдёүз§Қеёёи§Ғзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢпјҢе®ғд»¬з”ЁдәҺз®ЎзҗҶPodзҡ„йғЁзҪІе’Ңз”ҹе‘Ҫе‘ЁжңҹгҖӮжң¬ж–Үе°Ҷд»Ӣз»Қиҝҷдёүз§Қе·ҘдҪңиҙҹиҪҪзҡ„еҹәжң¬жҰӮеҝөгҖҒзү№зӮ№гҖҒйҖӮз”ЁеңәжҷҜпјҢе№¶еҜ№е®ғд»¬зҡ„ејӮеҗҢиҝӣиЎҢиҜҰз»ҶеҲҶжһҗгҖӮ

рҹ–јпёҸиғҢжҷҜ

еңЁе®№еҷЁзј–жҺ’дёӯпјҢеҰӮдҪ•й«ҳж•Ҳең°з®ЎзҗҶе’Ңи°ғеәҰPodжҳҜдёҖдёӘж ёеҝғй—®йўҳгҖӮKubernetesжҸҗдҫӣдәҶеӨҡз§Қе·ҘдҪңиҙҹиҪҪзұ»еһӢжқҘж»Ўи¶ідёҚеҗҢзҡ„йңҖжұӮпјҢе…¶дёӯDeploymentгҖҒStatefulSetе’ҢDaemonSetжҳҜжңҖеёёз”Ёзҡ„дёүз§Қзұ»еһӢгҖӮзҗҶи§Је®ғд»¬зҡ„зү№зӮ№е’ҢйҖӮз”ЁеңәжҷҜеҜ№дәҺи®ҫи®Ўе’ҢдјҳеҢ–KubernetesйӣҶзҫӨиҮіе…ійҮҚиҰҒгҖӮ

рҹ§ жҖқи·Ҝ

  1. д»Ӣз»ҚDeploymentгҖҒStatefulSetе’ҢDaemonSetзҡ„еҹәжң¬жҰӮеҝөгҖӮ
  2. жҺўи®ЁжҜҸз§Қе·ҘдҪңиҙҹиҪҪзҡ„зү№зӮ№е’ҢйҖӮз”ЁеңәжҷҜгҖӮ
  3. жҜ”иҫғе®ғд»¬д№Ӣй—ҙзҡ„ејӮеҗҢгҖӮ
  4. жҖ»з»“еҰӮдҪ•ж №жҚ®е®һйҷ…йңҖжұӮйҖүжӢ©еҗҲйҖӮзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢгҖӮ

рҹ”Ёи§ЈеҶі

жҖ»з»“иҖҢиЁҖпјҢDaemonSetжҳҜдёҖзұ»пјҢDeploymentе’ҢStatefulSetжҳҜдёҖзұ»гҖӮ

  • DaemonSetдёҚе…іеҝғPodеүҜжң¬ж•°йҮҸпјҢеҸӘе…іеҝғPodеңЁжҜҸдёӘиҠӮзӮ№дёҠзҡ„иҝҗиЎҢжғ…еҶөгҖӮ[5]дё“з”ЁдәҺйғЁзҪІе®ҲжҠӨиҝӣзЁӢпјҢдёҺиҠӮзӮ№з»‘е®ҡпјҢзЎ®дҝқжҜҸдёӘиҠӮзӮ№дёҠйғҪжңүеҜ№еә”зҡ„е®ҲжҠӨиҝӣзЁӢеүҜжң¬иҝҗиЎҢгҖӮеҜ№дәҺйңҖиҰҒеңЁдёҖжү№иҠӮзӮ№дёҠиҝҗиЎҢдё”ж— йңҖз”ЁжҲ·е№Ійў„зҡ„жҢҒз»ӯжҖ§еҗҺеҸ°д»»еҠЎйқһеёёжңүз”ЁгҖӮеә”з”ЁеңәжҷҜпјҡ
    • йӣҶзҫӨеӯҳеӮЁе®ҲжҠӨзЁӢеәҸпјҢеҰӮ glusterd гҖҒ ceph иҰҒйғЁзҪІеңЁжҜҸдёӘиҠӮзӮ№дёҠд»ҘжҸҗдҫӣжҢҒд№…жҖ§еӯҳеӮЁпјӣ
    • иҠӮзӮ№зӣ‘и§Ҷе®ҲжҠӨиҝӣзЁӢпјҢеҰӮ Prometheus зӣ‘жҺ§йӣҶзҫӨпјҢеҸҜд»ҘеңЁжҜҸдёӘиҠӮзӮ№дёҠиҝҗиЎҢдёҖдёӘ node-exporter иҝӣзЁӢжқҘж”¶йӣҶзӣ‘жҺ§иҠӮзӮ№зҡ„дҝЎжҒҜпјӣ
    • ж—Ҙеҝ—ж”¶йӣҶе®ҲжҠӨзЁӢеәҸпјҢеҰӮ fluentd жҲ– logstash пјҢеңЁжҜҸдёӘиҠӮзӮ№дёҠиҝҗиЎҢд»Ҙж”¶йӣҶе®№еҷЁзҡ„ж—Ҙеҝ—
  • Deploymentе’ҢStatefulSetеҲҶеҲ«дё“з”ЁдәҺйғЁзҪІ ==ж— зҠ¶жҖҒжңҚеҠЎ== е’Ң ==жңүзҠ¶жҖҒжңҚеҠЎ== гҖӮжүҖи°“зҡ„вҖңжңүзҠ¶жҖҒвҖқпјҢдё»иҰҒжҳҜй’ҲеҜ№PodеҸ‘з”ҹre-scheduleеҗҺд»Қз„¶иҰҒдҝқжҢҒд№ӢеүҚзҡ„зҠ¶жҖҒпјҲзҪ‘з»ңж ҮиҜҶ/жҢҒд№…еҢ–еӯҳеӮЁпјүпјҢж №жҚ®е®ҳж–№е»әи®®жҳҜж»Ўи¶ід»ҘдёӢд»»дёҖйңҖжұӮпјҡ[5]
    • зЁіе®ҡзҡ„гҖҒе”ҜдёҖзҡ„зҪ‘з»ңж ҮиҜҶгҖӮ
    • зЁіе®ҡзҡ„гҖҒжҢҒд№…зҡ„еӯҳеӮЁгҖӮ
    • жңүеәҸзҡ„гҖҒдјҳйӣ…зҡ„йғЁзҪІе’Ңдјёзј©гҖӮ
    • жңүеәҸзҡ„гҖҒдјҳйӣ…зҡ„еҲ йҷӨе’ҢеҒңжӯўгҖӮ
    • жңүеәҸзҡ„гҖҒиҮӘеҠЁзҡ„ж»ҡеҠЁжӣҙж–°гҖӮ

Deployment

жҰӮеҝө

Deployment жҳҜ Kubernetes дёӯжңҖеёёз”Ёзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢпјҢз”ЁдәҺз®ЎзҗҶж— зҠ¶жҖҒеә”з”Ёзҡ„йғЁзҪІе’Ңз”ҹе‘Ҫе‘ЁжңҹгҖӮе®ғйҖҡиҝҮеЈ°жҳҺејҸзҡ„ж–№ејҸе®ҡд№үPodзҡ„еүҜжң¬ж•°гҖҒжӣҙж–°зӯ–з•ҘзӯүгҖӮ

зү№зӮ№

  • ж— зҠ¶жҖҒпјҡйҖӮз”ЁдәҺж— зҠ¶жҖҒеә”з”ЁпјҢдҫӢеҰӮWebжңҚеҠЎгҖӮ
  • ж»ҡеҠЁжӣҙж–°пјҡж”ҜжҢҒйӣ¶еҒңжңәж—¶й—ҙзҡ„ж»ҡеҠЁжӣҙж–°е’Ңеӣһж»ҡгҖӮ
  • еүҜжң¬з®ЎзҗҶпјҡеҸҜд»ҘиҪ»жқҫжү©еұ•жҲ–зј©еҮҸPodзҡ„еүҜжң¬ж•°гҖӮ

йҖӮз”ЁеңәжҷҜ

  • WebжңҚеҠЎгҖҒAPIжңҚеҠЎзӯүж— зҠ¶жҖҒеә”з”ЁгҖӮ
  • йңҖиҰҒйў‘з№Ғжӣҙж–°зҡ„еә”з”ЁгҖӮ

StatefulSet

жҰӮеҝө

StatefulSet жҳҜ Kubernetes дёӯз”ЁдәҺз®ЎзҗҶжңүзҠ¶жҖҒеә”з”Ёзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢгҖӮе®ғзЎ®дҝқPodзҡ„йғЁзҪІйЎәеәҸгҖҒзҪ‘з»ңж ҮиҜҶе’ҢеӯҳеӮЁжҳҜзЁіе®ҡзҡ„гҖӮ

зү№зӮ№

  • жңүзҠ¶жҖҒпјҡжҜҸдёӘPodйғҪжңүе”ҜдёҖзҡ„ж ҮиҜҶпјҲеҰӮpod-0гҖҒpod-1пјүгҖӮ
  • зЁіе®ҡзҡ„зҪ‘з»ңж ҮиҜҶпјҡжҜҸдёӘPodйғҪжңүеӣәе®ҡзҡ„DNSеҗҚз§°гҖӮ
  • жҢҒд№…еҢ–еӯҳеӮЁпјҡж”ҜжҢҒдёәжҜҸдёӘPodеҲҶй…ҚзӢ¬з«Ӣзҡ„жҢҒд№…еҢ–еӯҳеӮЁеҚ·гҖӮ

йҖӮз”ЁеңәжҷҜ

  • ж•°жҚ®еә“пјҲеҰӮMySQLгҖҒPostgreSQLпјүгҖӮ
  • еҲҶеёғејҸзі»з»ҹпјҲеҰӮZooKeeperгҖҒKafkaпјүгҖӮ

DaemonSet

жҰӮеҝө

DaemonSet жҳҜ Kubernetes дёӯз”ЁдәҺзЎ®дҝқжҜҸдёӘиҠӮзӮ№дёҠиҝҗиЎҢдёҖдёӘPodзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢгҖӮе®ғйҖҡеёёз”ЁдәҺиҝҗиЎҢйӣҶзҫӨиҢғеӣҙзҡ„еҗҺеҸ°д»»еҠЎгҖӮ

зү№зӮ№

  • жҜҸиҠӮзӮ№дёҖдёӘPodпјҡзЎ®дҝқжҜҸдёӘиҠӮзӮ№дёҠйғҪжңүдёҖдёӘPodиҝҗиЎҢгҖӮ
  • иҮӘеҠЁйҖӮй…Қж–°иҠӮзӮ№пјҡеҪ“ж–°иҠӮзӮ№еҠ е…ҘйӣҶзҫӨж—¶пјҢDaemonSetдјҡиҮӘеҠЁеңЁж–°иҠӮзӮ№дёҠеҲӣе»әPodгҖӮ
  • йҖӮз”ЁдәҺеҗҺеҸ°д»»еҠЎпјҡеҰӮж—Ҙеҝ—ж”¶йӣҶгҖҒзӣ‘жҺ§д»ЈзҗҶзӯүгҖӮ

йҖӮз”ЁеңәжҷҜ

  • ж—Ҙеҝ—ж”¶йӣҶпјҲеҰӮFluentdпјүгҖӮ
  • зӣ‘жҺ§д»ЈзҗҶпјҲеҰӮPrometheus Node ExporterпјүгҖӮ
  • зҪ‘з»ңжҸ’д»¶пјҲеҰӮCalicoгҖҒFlannelпјүгҖӮ

DeploymentгҖҒStatefulSetе’ҢDaemonSetзҡ„ејӮеҗҢ

зү№жҖ§DeploymentStatefulSetDaemonSet
зҠ¶жҖҒз®ЎзҗҶж— зҠ¶жҖҒжңүзҠ¶жҖҒж— зҠ¶жҖҒ
Podж ҮиҜҶдёҚеӣәе®ҡеӣәе®ҡдёҚеӣәе®ҡ
еӯҳеӮЁдёҚж”ҜжҢҒжҢҒд№…еҢ–еӯҳеӮЁж”ҜжҢҒжҢҒд№…еҢ–еӯҳеӮЁдёҚж”ҜжҢҒжҢҒд№…еҢ–еӯҳеӮЁ
йҖӮз”ЁеңәжҷҜWebжңҚеҠЎгҖҒAPIжңҚеҠЎж•°жҚ®еә“гҖҒеҲҶеёғејҸзі»з»ҹж—Ҙеҝ—ж”¶йӣҶгҖҒзӣ‘жҺ§д»ЈзҗҶ
PodеҲҶеёғж №жҚ®еүҜжң¬ж•°еҲҶеёғж №жҚ®еүҜжң¬ж•°еҲҶеёғжҜҸиҠӮзӮ№дёҖдёӘPod
жӣҙж–°ж–№ејҸж»ҡеҠЁжӣҙж–°ж»ҡеҠЁжӣҙж–°жүӢеҠЁжӣҙж–°

рҹҸҘеҸҚжҖқ

йҖҡиҝҮеҜ№DeploymentгҖҒStatefulSetе’ҢDaemonSetзҡ„еӯҰд№ пјҢжҲ‘еҜ№KubernetesдёӯдёҚеҗҢе·ҘдҪңиҙҹиҪҪзҡ„зү№зӮ№е’ҢйҖӮз”ЁеңәжҷҜжңүдәҶжӣҙж·ұе…Ҙзҡ„зҗҶи§ЈгҖӮеңЁе®һйҷ…йЎ№зӣ®дёӯпјҢйҖүжӢ©еҗҲйҖӮзҡ„е·ҘдҪңиҙҹиҪҪзұ»еһӢеҸҜд»Ҙжҳҫи‘—жҸҗй«ҳзі»з»ҹзҡ„еҸҜйқ жҖ§е’ҢеҸҜз»ҙжҠӨжҖ§гҖӮдҫӢеҰӮпјҢеҜ№дәҺж— зҠ¶жҖҒзҡ„WebжңҚеҠЎпјҢеҸҜд»ҘйҖүжӢ©DeploymentпјӣеҜ№дәҺйңҖиҰҒжҢҒд№…еҢ–еӯҳеӮЁзҡ„ж•°жҚ®еә“пјҢеҸҜд»ҘйҖүжӢ©StatefulSetпјӣиҖҢеҜ№дәҺйңҖиҰҒеңЁжҜҸдёӘиҠӮзӮ№дёҠиҝҗиЎҢзҡ„еҗҺеҸ°д»»еҠЎпјҢеҲҷеҸҜд»ҘйҖүжӢ©DaemonSetгҖӮ

еңЁжңӘжқҘзҡ„е·ҘдҪңдёӯпјҢжҲ‘дјҡе°қиҜ•з»ҷеҮәдёүз§Қж–№жЎҲзҡ„е…·дҪ“д»Јз ҒжЎҲдҫӢ[4]пјҢжңҖеҘҪиғҪеӨҹжүҫеҲ°еҗҲйҖӮзҡ„еңәжҷҜе°Ҷе…¶дҪңдёәе®һйҷ…еә”з”ЁпјҲд№ҹи®ёжүҫдәӣйқўз»ҸдјҡжҳҜдёҚй”ҷзҡ„ж–№еҗ‘пјүгҖӮд№ҹдјҡиҖғиҷ‘з»§з»ӯдәҶи§Је…¶д»–з»„д»¶пјҢеҰӮReplication ControllerпјҲRCпјүе’ҢReplicaSetпјҲRSпјү[6]гҖӮ

  • еёҢжңӣиҝҷзҜҮеҚҡе®ўеҜ№дҪ жңүеё®еҠ©пјҒеҰӮжһңдҪ жңүд»»дҪ•й—®йўҳжҲ–йңҖиҰҒиҝӣдёҖжӯҘзҡ„её®еҠ©пјҢиҜ·йҡҸж—¶жҸҗй—®гҖӮ
  • еҰӮжһңдҪ е–ңж¬ўиҝҷзҜҮж–Үз« пјҢж¬ўиҝҺеҠЁеҠЁе°ҸжүӢз»ҷжҲ‘дёҖдёӘfollowжҲ–starгҖӮ
